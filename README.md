# Aura-Server

## Packages to install:
#### 1. Gin Web Framework 
```bash
go get -u github.com/gin-gonic/gin
```
#### 2. Gorm 
```bash
go get -u gorm.io/gorm
```
#### 3. CompileDeamon
```bash
go get github.com/githubnemo/CompileDaemon
```

---
## Command to run
```bash
~/go/bin/CompileDaemon -command="./Aura-Server"
```
